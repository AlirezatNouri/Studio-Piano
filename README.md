# Studio Piano

Studio Piano is a browser-based piano simulator built with plain HTML, CSS, and JavaScript.

## Features

- Full piano range from `A0` to `C8`
- Mouse and computer-key input
- Adjustable keyboard window across the instrument
- Volume, decay, and tone controls
- Panic button to stop all active notes
- Responsive static frontend with no build step

## Run Locally

Open `index.html` in a modern browser.
